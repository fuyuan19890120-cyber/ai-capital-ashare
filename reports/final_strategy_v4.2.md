# 策略一 V4.2 定稿方案

> 2026-07-19 定稿 | 基于 V4.1 + 反转硬过滤
> V4.1 定稿: reports/final_strategy_v4.1.md
> 改进动机: 主动基金经理策略调研 → 制度切换权重试验(RC-FW v1/v2) → 发现反转过滤单独贡献 +5.2pp
> 完整试验记录: backtests/rcfw_v1_results.json, backtests/rcfw_v2_results.json

## 一、V4.1 → V4.2 的唯一改动

**加了一个过滤条件**：选股前剔除近 20 日涨幅排名前 25% 的候选股。

```
V4.1 选股:
  指数成分池(~380只) → 低波+动量排名 → Top-15

V4.2 选股:
  指数成分池(~380只) → 剔除近20日涨幅前25% → 低波+动量排名 → Top-15
```

其他一切不变：制度择时(SMA250四档+SURGE)、因子定义(低波43%+动量45%)、执行(T+1开盘、全成本)、风控(板块40%、个股15%)。

**为什么有效**：A股截面动量长期为负——追涨短期涨幅最高的股票几乎必亏（JFE 2019 CH-3/CH-4、1296组动量参数实证）。反转过滤把这个最可靠的量价信号用作剔除项，堵住了现有因子体系最大的漏洞。

## 三、改进归因

```
V4.1 基线:                              11.7%
  + 反转过滤(剔除近20日涨幅前25%):        +5.2pp  ← 全部改善
  + 因子计算差异:                         -0.1pp  ← 可忽略
  = V4.2:                                16.8%*
  + 制度切换权重(RISKON加重动量):         -0.9pp  ← 负贡献, 否决
  = 最终 V4.2:                           15.9%**
  * rcfw_v2_fixed (NEUTRAL固定权重=逼近V4.1, 但用rcfw因子计算管线)
  ** run_final_backtest --mode fixed 直接产出 18.3%(全缓存池, 两个数字跨管线口径差异~2pp)
```

**结论**：反转过滤独自贡献了全部改善。制度切换因子权重是负贡献（−0.9pp），否决。V4.2 就是 V4.1 + 一行过滤——最简单的改动，最大的效果。

## 四、RC-FW v1/v2 试验存档

### RC-FW v1（真实因子池，制度切换）
- 因子池：低波250 / 夏普动量 / PMO / ROE质量 / EP（复用 V5 管线）
- 制度切换效应：+1.0pp（rcfw_45 6.5% vs fixed 5.5%）
- 但整体跑输 V4.1 5.2pp——差距全部来自 2025-26 时段（真因子在该段踏空高贝塔行情）
- 动量 35% 全面优于 45%（更高收益、更低回撤、更高夏普）

### RC-FW v2（V4.1 简单因子，制度切换）
- 因子池：低波63d / 6月夏普动量 / 250日收益（与 V4.1 完全一致）
- 制度切换效应：-0.9pp（否决）
- 发现反转过滤单独贡献 +5.2pp
- 制度切换本身在 V4.1 因子上是负贡献——V4.1 的 43%/45% 混合已是各制度下的最优平衡

### 关键教训
1. **在 A 股大盘池里，简单价格因子（低波+动量）优于学术因子（PMO/ROE/EP）**——学术因子在 2025 高贝塔行情中系统性踏空
2. **反转过滤是唯一稳健的 alpha 增量**——A 股反转效应是最强量价因子，当剔除项用比当排名因子用好得多
3. **因子权重本身已是最优**——V4.1 的权重是样本内反复调试的结果，制度切换只增加了噪声

## 五、与主动基金经理策略的对照

| 来源 | 手法 | V4.2 采纳情况 |
|---|---|---|
| A股文献共识 | 1月反转是最强量价因子 | ✅ 反转过滤(剔除前25%) |
| 王平 | PB-ROE + 行业中性化 | ❌ 学术因子在大盘池无 alpha (RC-FW v1 验证) |
| 贾成东 | 顺大势逆小势(制度切换) | ❌ 制度切换因子权重负贡献 (RC-FW v2 验证) |
| 周文群 + 王平 | 现金流质量过滤 | 待后续（需要季报数据管线，成本高于收益证据尚不足） |
| 董辰 | 仓位灵活调整 | 已有四档制度仓位映射 + SURGE 加速，暂时够用 |

**核心结论**：主动基金经理的策略思想在量化转化时，最高效的不是复制其选股逻辑，而是用他们的思维方式审视现有策略的漏洞。反转过滤就是"贾成东的逆小势"的量化表达——在趋势中避免追高。

## 六、已知风险

1. 回撤 -26% 仍是常态，不是尾部——持有前提不变
2. 2021 年 +70.8% 包含极端行情（结构性牛市+反转过滤恰好完美匹配），不可当作常态预期
3. 反转过滤在急跌后反弹的市场中可能误杀（跌了 20 天刚反弹就被过滤），但 A 股急跌后罕见 V 型反转，误杀概率低
4. SURGE 收益集中于少数 episode（2019-02/2020-05/2024-09），假触发年份（2018/2023）真实亏钱
5. 全缓存池含幸存者偏差，可信口径需 P4 复测
