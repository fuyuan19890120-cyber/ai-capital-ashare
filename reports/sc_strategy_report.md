# 中小盘股量化策略 V4.2-SC 定稿

> 2026-07-20 | CSI500+CSI1000 宇宙 | liq(50%)+mom(50%) | 无择时 | 安全过滤

## 一、策略定义

### 核心规则

| 维度 | 规定 |
|------|------|
| 宇宙 | CSI500 + CSI1000 成分股（缓存 833 只） |
| 因子 | 流动性 50% + 动量 50% |
| 过滤 | ST → 反转 20% → pbMRQ>0 → 股价≥1.5元 |
| 选股 | Top-30，**双月调仓**，等权 |
| 择时 | **无**（永远满仓 95% 股票） |
| 执行 | T+1 开盘成交，全成本（佣金+印花税+滑点） |
| 涨跌停 | 涨停不追，跌停不卖，停牌 ffill 估值 |

### 因子含义

**流动性因子**（权重 50%）：
- PMO = sum(turn,20)/sum(turn,250)，越低越好 — 近一月没被炒作的票
- Amihud = mean(|ret|/amount, 20)，越高越好 — 难买到的票有流动性补偿溢价
- 涨跌停日(|ret|≥9.5%)不参与 Amihud 计算
- 本质：选"被遗忘的好股票"

**动量因子**（权重 50%）：
- 夏普动量 = ret(T-120→T-20) / std(窗口日收益)，越高越好
- MAX20 彩票中性化：近 20 日最高收益前 20% 的票，动量分强制记 0.5
- 本质：选"正在稳步上涨的股票"，不是追涨

**两因子结合**：选"正在从被遗忘状态中慢慢走出来的股票"——流动性确保低估，动量确保不是价值陷阱。

### 安全过滤

| # | 过滤 | 数据源 | 防什么 |
|---|------|------|------|
| 1 | isST == 1 剔除 | Baostock 日频 | 已 ST 的股票 |
| 2 | 近 20 日涨幅前 20% 剔除 | 日线 close | A 股反转效应，追涨必亏 |
| 3 | pbMRQ ≤ 0 剔除 | Baostock 日频 | 资不抵债，破产风险 |
| 4 | 收盘价 < 1.5 剔除 | 日线 close | 面值退市预警 |
| 5 | ≥250 个交易日 | 日线 | 次新股数据不足 |

## 三、研究历程

### 否决了的东西

| 尝试 | 结果 | 原因 |
|------|:---:|------|
| 制度择时（SMA250 × SURGE） | ❌ -11pp | 中小盘 alpha 和沪深 300 脱钩，择时砍掉的是收益不是风险 |
| 低波动率因子 | ❌ -17pp | 选出的"低波小盘"是无 alpha 的死股 |
| EP 价值因子 | ❌ -19pp | A 股 EP 在小盘上无区分力 |
| 质量因子（ROE+ΔROE+稳定性） | ❌ -6pp | 小盘质量信号噪声大 |
| 五因子全集 | ❌ -16pp | 稀释了 liq+mom 的 alpha |
| IVOL（特质波动率） | ❌ -3.2pp | 踢掉的"高特质波动"恰是 alpha 来源 |
| 股价≥3 元过滤 | ❌ -0.8pp | 踢掉的超低价股含反转候选 |
| 成交额下限过滤 | ❌ -0.3pp | 踢掉的微盘股贡献流动性溢价 |
| 月频以上调仓 | ❌ -2pp | 信号半衰期长，频繁调仓 = 噪声 |

## 四、与 V4.2 大盘策略的关系

| | V4.2 大盘 | V4.2-SC 中小盘 |
|------|:---:|:---:|
| 年化（全缓存） | ~22% | 21.98% |
| 回撤 | -25.7% | -31.1% |
| Sharpe | 1.13 | 1.06 |
| 因子 | 2 个价格因子 | 2 个量价因子 |
| 择时 | SMA250 × SURGE | **无** |
| 核心 alpha 来源 | 择时 + 反转过滤 | 因子选股（liq+mom） |
| 安全过滤 | ST + 反转 | ST + 反转 + pbMRQ + 价格 |

两个策略年化相当，互补性强：大盘靠择时控制回撤，中小盘靠因子选股创造 alpha。

## 五、已知风险

1. **幸存者偏差**：全缓存池无退市股，美化估计 7-15pp。可信年化约 10-15%
2. **缓存覆盖率**：833/1201（69.4%），缺失 368 只
3. **回撤 -31%**：2024 年初微盘暴跌是最大考验，实盘能否扛住
4. **流动性风险**：Amihud 因子倾向选低流动性股票，极端行情可能无法平仓
5. **容量**：中小盘约 50-100 万 CNY
6. **因子失效**：低换手+动量在 A 股不是新因子，可能因拥挤而衰减
